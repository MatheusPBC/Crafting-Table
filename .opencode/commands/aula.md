Prepare uma aula tecnica sobre `$ARGUMENTS` usando o formato de aula do projeto.

Regras obrigatórias:

- Responda em português.
- Use a skill `aula-md` quando disponível.
- Só crie arquivo Markdown porque este comando foi explicitamente chamado para isso.
- Se o tópico envolver código do projeto, pesquise primeiro os arquivos relevantes e cite fontes consultadas.
- Não execute comandos sem confirmação explícita do usuário.

Entregue ou gere conforme combinado:

- Introdução e objetivo prático.
- Quando usar e quando não usar.
- Pré-requisitos.
- Conceitos fundamentais.
- Arquitetura e fluxo com Mermaid.
- Passo a passo aplicável.
- Exemplos comentados.
- Troubleshooting.
- Boas práticas, limites, checklist, exercícios e FAQ.
